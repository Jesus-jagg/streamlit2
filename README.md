# 🌟 Dashboard Sector Minero Energético Colombia

## Descripción

Aplicación interactiva desarrollada con Streamlit para visualizar y analizar datos del sector minero-energético de Colombia. Incluye métricas de eficiencia energética, proyectos, inversiones, equipos de investigación y recursos minerales.

## 🚀 Características

- Dashboard principal con KPIs clave
- Análisis de eficiencia energética por tipo de energía
- Visualización de proyectos activos
- Seguimiento de inversiones
- Gestión de equipos de investigación
- Mapeo de recursos minerales

## 📋 Requisitos

- Python 3.8+
- Streamlit
- Pandas
- Plotly

## 🔧 Instalación

```bash
# Clonar el repositorio
git clone <repo-url>
cd sector-minero-energetico

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar la aplicación
streamlit run app.py
````
